# CreateSeal
电子印章生成工具。中间的文字可以自定义包括特殊字符，比如五角星(☆★)党徽()之类的。复杂一点的可以支持选择选择图片生成印章。
这个最初版是参考的http://www.csharpwin.com/csharpspace/472.shtml ，然后借鉴了一个csdn上的源码，自己做的改动就是：
1.加了生成图片的页面，方便在页面上调用，使用方法和验证码一样。
2.添加了功能，使中间的内容可以使用图片。上述的方法，都不能实现复杂的印章中部内容，比如中华人民共和国国徽之类的图案。
